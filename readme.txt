Это реализация-"велосипед" программы для командной строки, которая
при своем запуске выдает на stdout текущую дату и время в формате:
YYYYMMDD_hhmmss
Используется в программе смены аккаунта в MMORPG Аллоды Онлайн.
Эта реализация написана, чтоб не болела голова про всякие лицензии.

Скомпилировано в FreePascal compiler v3.0.0
